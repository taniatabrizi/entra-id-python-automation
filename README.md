# Entra ID Python Automation

Python-based IAM automation project using Microsoft Graph API to perform identity management tasks.

## Purpose

Demonstrates identity engineering concepts including:

- User lifecycle automation
- Identity reporting
- Group management
- Access governance workflows
- Microsoft Graph API integration

## Technologies

- Python
- Microsoft Graph API
- OAuth 2.0
- Microsoft Entra ID
- REST APIs

## Planned Features

- Retrieve Entra ID users
- Generate identity reports
- Identify inactive accounts
- Automate group membership workflows
- Export IAM audit data
